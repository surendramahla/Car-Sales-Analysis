Project Overview

This project focuses on analyzing automobile sales data to understand sales performance, pricing trends, and customer preferences. The dashboard provides key insights using interactive visualizations to support data-driven decision making.

Objectives :

  Analyze YTD, MTD, and YOY car sales performance

  Identify trends based on body style, color, and time

  Track average price and cars sold metrics

  Support strategic decisions in sales, pricing, and inventory

Dataset :

  Automobile sales dataset containing transaction-level data

  Key fields include sales date, car model, body style, color, price, and quantity sold

Tools & Technologies : 

  Power BI – Dashboard design and DAX calculations

  Excel / CSV – Data source

  SQL / Python (optional) – Data cleaning and analysis

Key KPIs :

  Year-to-Date (YTD) Total Sales

  Month-to-Date (MTD) Total Sales

  Year-over-Year (YOY) Growth

  Average Selling Price

  Total Cars Sold

  Insights Generated

  Identified top-performing car body styles and colors

  Observed seasonal sales trends and growth patterns

Analyzed price variations impacting overall sales performance

Conclusion

This dashboard enables stakeholders to monitor sales performance, compare trends over time, and make informed business decisions using data-driven insights.
